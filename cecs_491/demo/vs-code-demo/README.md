# Visual Studio Code Demo

## About

This demo shows how to utilize VS Code for modern web development.  Primarily focuses on the IDE capabilities such as debugging, extensions and the node package manager (npm).  In addition, this project contains an example of how to get started with nodeJS and ExpressJS.

## Project Root

* package.json
  * npm specific file to handle project dependencies

* index.html
  * Main entry point into demo

* geo-models.js
  * Example of leveraging JavaScript language constructs to model POCO objects

## Sub-directories

* .vscode
  * VS Code specific folder for storing IDE settings

* express
  * views
    * Example of organizing HTML as separate "views"
  * express-setup.js
    * Example of the boilerplate code needed to start up ExpressJS
